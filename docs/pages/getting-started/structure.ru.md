# Структура

## Синтаксис

После создания экземпляра `PrismaClient` запросы могут быть выполнены следующим образом:

```rust
client
    .post() // Модель для запроса
    .find_many(vec![]) // Запрос для выполнения
    .exec() // Завершение запроса
    .await; // Все запросы асинхронные и возвращают Result
```

Запросы могут быть отфильтрованы и расширены с помощью сгенерированных модификаторов.
Каждая модель в вашей схеме получает соответствующий модуль Rust, с соответствующими модулями для их полей внутри.
Полевые модули (Field modules) содержат функции для создания модификаторов на основе каждого поля.

```rust
use prisma::post;

client
    .post()
    .find_many(vec![
//      model::
//            field::
//                   method()
        post::title::equals("Test".to_string())
    ])
    .exec()
    .await;
```

Все модели и имена полей преобразуются в `snake_case`, чтобы соответствовать соглашениям по именованиям Rust.
